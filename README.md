# Blog SEMANA — Technical Test

## 🎨 UI Inspiration
Diseño inspirado en la revista Semana Colombia.
Paleta: Rojo #CC0000, Negro, Blanco — tipografía editorial.

## ✅ Features
- 📰 Listado de posts con paginación
- 👤 Sidebar con filtro por autor
- 🏷️ Vista de tags con filtro por tag
- 💬 Detalle de post por URL con comentarios y reacciones
- 🔐 Google Sign-In (Firebase) — ruta protegida
- 👥 Usuarios con foto (solo autenticados)

## 🛠 Tech Stack
React 18 · Vite · React Router v6 · TanStack Query v5 · Zustand · Firebase Auth · Tailwind CSS

## 🚀 Ejecutar localmente
cp .env.example .env
# Completar credenciales Firebase en .env
npm install
npm run dev

## 🌐 Deploy
npm run deploy
